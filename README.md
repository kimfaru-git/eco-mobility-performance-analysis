# Eco-Mobility Performance Analysis
친환경·모빌리티 차량 성능 데이터를 활용하여  
연비, 배출가스, 차량 스펙 등을 분석하고, 이를 기반으로  
효율적인 차량 설계 전략을 제안하는 데이터 분석 프로젝트입니다.

---

## 📌 Project Goal
- 차량의 엔진/차체/연료 특성이 연비와 CO₂ 배출량에 미치는 영향 분석
- 친환경 차량 성능을 데이터 기반으로 비교 및 효율적인 설계 전략 도출
- 활용 데이터: `차량_연비_데이터(20022022).csv`

---
